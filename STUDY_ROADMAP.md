# 🗺️ SQL Study Roadmap for Data Scientists

A structured week-by-week plan to go from zero to interview-ready.

---

## 🟢 Week 1 — Foundations
- [ ] SELECT, FROM, WHERE, ORDER BY, LIMIT
- [ ] AND, OR, NOT, IN, BETWEEN, LIKE
- [ ] IS NULL, IS NOT NULL, COALESCE
- [ ] Basic math: +, -, *, /, ROUND, ABS

**Practice file:** `sql_for_data_science.py` → Sections 1 & 2

---

## 🟡 Week 2 — Aggregations & Joins
- [ ] COUNT, SUM, AVG, MIN, MAX
- [ ] GROUP BY, HAVING
- [ ] INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN
- [ ] Self JOIN (employees ↔ managers)

**Practice file:** `sql_for_data_science.py` → Sections 3 & 4

---

## 🟠 Week 3 — Intermediate
- [ ] Subqueries (WHERE, FROM, SELECT)
- [ ] CTEs (WITH clause)
- [ ] CASE WHEN / CASE THEN ELSE
- [ ] UNION, INTERSECT, EXCEPT
- [ ] String functions: UPPER, LOWER, LENGTH, SUBSTR, TRIM
- [ ] Date functions: SUBSTR, strftime (SQLite), DATE_FORMAT (MySQL)

**Practice file:** `sql_for_data_science.py` → Sections 5, 6, 7, 8 & 9

---

## 🔴 Week 4 — Advanced (Interview Level)
- [ ] Window functions: ROW_NUMBER, RANK, DENSE_RANK
- [ ] Window functions: SUM OVER, AVG OVER, LAG, LEAD
- [ ] PARTITION BY vs ORDER BY in windows
- [ ] PIVOT using CASE WHEN
- [ ] Self JOINs for hierarchical data
- [ ] Gaps and islands problems
- [ ] Retention / cohort analysis patterns

**Practice file:** `sql_interview_questions.py` → Medium & Hard sections

---

## 📊 Week 5 — Pandas ↔ SQL Fluency
- [ ] Translate every SQL operation to Pandas and back
- [ ] Know when to use SQL vs Pandas vs both

**Practice file:** `pandas_vs_sql.py`

---

## 🎯 Interview Prep Checklist

- [ ] Second highest / Nth highest salary
- [ ] Duplicates detection and removal
- [ ] Top N per group
- [ ] Month-over-month / YoY growth
- [ ] Retention / churn analysis
- [ ] Self JOIN (manager vs employee)
- [ ] Running totals and moving averages
- [ ] PIVOT table using CASE WHEN
- [ ] Consecutive day streaks (gaps & islands)
- [ ] Customers active in period A but not B
